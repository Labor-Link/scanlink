.NET Framework

1. Supported Platforms

  1. Windows

2. Requirements

  1. Requires Microsoft Visual Studio 2008 or later in order to compile/execute the sample programs in the ARGOX SDK.
  2. Requires Windows XP or later.
  3. Requires Microsoft .NET Framework v3.5 Service Pack 1: Install Windows .NET Framework 3.5 Service Pack 1

3. Supported Functions

  1. Supported Emulators
    • PPLB
    • PPLZ
  2. Supported Transmission Interface
    • File
    • Serial Port
    • Parallel Port
    • USB
    • TCP/IP

4. Directory Contents

    /windows_desktop
    • /Doc => Related documentation
    • /Example => Sample programs, including C#
      • /src => Source code of sample program
      • /release => Executable file of the sample program
    • /Library => Class libraries and XML files for IntelliSense

5. Tips and Reminders

  1. To execute Argox's sample application or other applications developed with the ARGOX SDK, Microsoft .NET Framework v3.5 must be installed on the desktop system. If it is not installed, a message will be displayed to notify the user that the application requires Microsoft .NET Framework version to be installed on the desktop system.
  2. It is recommended to put the class library and application in the same directory to avoid issues with the application being unable to execute or unable to locate the class library.
  3. It is recommended to put the XML files and the class library in the same directory, so as to support the IntelliSense feature, which makes it more convenient to use this class library.
  4. If the execution result is different from what is expected, use the file transmission interface. You will get a command file that should be transmitted to the printer. Please use it to confirm whether the command is correct.
  5. If your desired transmission interface is not supported, use the file transmission interface. You will get a command file that should be transmitted to the printer, or use the function to get the byte[] command data. And then you can control the transmission interface, send the file content or the obtained byte[] command data through the transmission interface you control.

Java

1.1. Supported Platforms

  1. Android

1.2. Requirements

  1. Required Android Studio 3.4.1 or later in order to compile/execute the sample programs in the ARGOX SDK.
  2. Required Android 4.3 (API Level 19) or higher system version, then can execute the sample programs in the ARGOX SDK.
  3. USB connection requires mobile phone to support USB HOST function
  4. The bluetooth connection requires the mobile phone to support bluetooth 3.0 or higher

1.3. Supported Functions

    • PPLB
    • PPLZ
  2. Supported Transmission Interface

    • USB
    • TCP/IP
